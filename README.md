# EarthToRhino

![Rhino Spacesuit](/Assets/Rhino%20Spacesuit_Short.gif)

## What is Earth To Rhino
EarthToRhino is a bridge between Cesium, a 3d Geospatial platform 🌎, and McNeel's Rhinoceros 3D 🦏 through Grasshopper 🦗.
